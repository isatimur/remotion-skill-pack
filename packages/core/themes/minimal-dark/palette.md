# minimal-dark Palette

General-purpose dark theme for technical video presentations.

| Role | Value | Usage |
|------|-------|-------|
| background | `#111827` | Canvas fill |
| surface | `#1f2937` | Card / panel backgrounds |
| text | `#e5e7eb` | Primary readable text |
| textMuted | `#9ca3af` | Captions, annotations |
| accent | `#60a5fa` | Headlines, highlights |
| accentDim | `#2563eb` | Accent secondary |
| border | `#374151` | Dividers, card borders |
| code | `#0f172a` | Code block background |

## Animation

- Entry: 8 frames fade-in (spring, damping 18, stiffness 200)
- Transition scenes: 8 frames
- Bullet stagger: 5 frames
