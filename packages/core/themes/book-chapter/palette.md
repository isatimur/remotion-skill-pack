# book-chapter Palette

Theme for *From Copilot to Colleague* chapter videos.

| Role | Value | Usage |
|------|-------|-------|
| background | `#0f0f14` | Canvas fill |
| surface | `#1a1a24` | Card / panel backgrounds |
| text | `#f0f0f0` | Primary readable text |
| textMuted | `#a0a0b0` | Captions, annotations |
| accent | `#6ee7b7` | Headlines, highlights, progress bars |
| accentDim | `#2d9e7a` | Accent hover / secondary accent |
| border | `#2a2a3a` | Dividers, card borders |
| code | `#1e1e2e` | Code block background |

## Typography

- **Heading**: Inter (system-ui fallback)
- **Body**: Inter (system-ui fallback)
- **Mono**: JetBrains Mono → Fira Code → monospace

## Animation

- Entry: 15 frames fade-in + slide-up (spring, damping 20, stiffness 180)
- Transition scenes: 15 frames
- Bullet stagger: 8 frames between each bullet
