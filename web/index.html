Install in one command

Pick your AI agent. Run once from your project root.

AgentInstall commandWhat it does
Claude Codenpx @remotion-skill-pack/install claude-codeCopies SKILL.md to ~/.claude/skills/, registers MCP
Cursornpx @remotion-skill-pack/install cursorWrites .cursor/rules/remotion-skill-pack.mdc
GitHub Copilotnpx @remotion-skill-pack/install copilotMerges section into .github/copilot-instructions.md

CLI onlynpx @remotion-skill-pack/install cliCopies skill to .remotion-skill-pack/ locally

Themes

Two bundled themes. Build your own with npx @remotion-skill-pack/create-template.

book-chapter

Designed for From Copilot to Colleague chapter videos. Matches the Excalidraw stripe-press visual language.

minimal-dark

Clean neutral dark theme. Good for any general technical presentation.

MCP Tools

Five tools available when the MCP server is running.

create_composition

Generate a starter composition.json from a chapter title and key ideas.

validate_composition

Validate a composition JSON string against the schema.

render_to_tsx

Convert composition JSON to Scene.tsx, Root.tsx, and entry.tsx strings.

list_themes

List available themes with name, description, and full palette.

get_scene_template

Get example props and recommended duration for any scene type.
